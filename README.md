## 👋 About This Repo

This repository documents my complete AWS Solutions Architect Associate (SAA-C03) preparation journey — from day one to exam day.

Every day I study, I fill in a structured `.md` note file covering what I learned, key services, exam traps, and practice question mistakes. The goal is to:

- 📚 Build a personal revision library
- 💡 Reinforce learning by writing things in my own words
- 🗂️ Showcase consistent daily progress to recruiters
- 🔁 Have everything in one place for quick revision

## 📝 Daily Note Template

Every day file follows this structure:

```markdown
# Day XX — Topic Name

## What I Learned Today
## Key Services & Comparisons (table)
## Exam Traps & Gotchas
## Questions I Got Wrong
## Hands-On Done Today
## Resources Used
## One-Line Summary
```

> The **One-Line Summary** at the bottom of each file is the most important part.
> If I can't summarise the day in one sentence, I rewatch before moving on.

This repo includes a **GitHub-synced study tracker** built from scratch.

**Features:**
- ✅ Checkbox for every study day
- 📊 Progress bar per week
- ☁️ Saves progress to GitHub automatically (via GitHub API)
- 📱 Works from any device, any browser
- 🔒 Progress is private — only you can save with your token

### Use this tracker yourself

1. **Fork** this repo
2. Go to **Settings → Pages → main → root** → Save
3. Open your Pages URL
4. Fill in your GitHub username, repo name, and Personal Access Token
5. Start ticking! ✅
